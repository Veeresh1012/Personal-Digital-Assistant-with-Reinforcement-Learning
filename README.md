# Personal Digital Assistant with Reinforcement Learning

## Overview
This project is a personal digital assistant system that uses reinforcement learning techniques to improve user interaction and automate tasks. It consists of a mobile application, backend assistant logic, and a web dashboard.

## Features
- Voice input-based interaction
- Task automation
- Reinforcement learning-based decision making
- Mobile application interface
- Web dashboard for monitoring and control

## Tech Stack
- Python (Machine Learning / Reinforcement Learning)
- React / React Native
- TypeScript
- Web technologies

## Project Structure
- MyMobileApp/ → Mobile application (React Native)
- my-personal-assistant/ → Core assistant logic
- web_dashboard/ → Dashboard interface
